# LSJS-Quick-Reference muss konsultiert werden

ctx: LSJS-Module, Views, Controller, Models, `app.js`
anlegen | bearbeiten.

## Anweisungen

- Vor jeder LSJS-Änderung:
  `assets/lsjs/QUICKREF_AI.md` im Projekt-Workspace
  lokalisieren, lesen, Konventionen befolgen.
- miss:`QUICKREF_AI.md` => Operator bitten, Datei
  bereitzustellen; => !LSJS-Änderungen vor Konsultation.
- Implementierung: den Mustern der Quick-Reference folgen.
- Quick-Reference ungenau | unvollständig => Operator
  informieren & konkrete Verbesserungen vorschlagen;
  => !Änderungen auf unklarer Vorgabe ohne Bestätigung.
